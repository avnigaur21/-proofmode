# ProofMode Project Brief

ProofMode is an independent proof layer for AI coding agents.

The core product question is:

```txt
The agent says the task is done. What evidence proves that?
```

## Current Differentiator

ProofMode is not only checking code. It checks the relationship between:

- the agent completion claim
- the agent self-report
- UI/API/DB/Git evidence
- evaluator verdict
- human approval

## Current Product Surface

- Dashboard for human review
- CLI for local and CI workflows
- GitHub PR workflow
- Evidence bundle export
- Saved project profiles
- JSON and Markdown artifacts

## Strongest Interview Framing

ProofMode turns AI coding-agent completion claims into auditable evidence packages with deterministic guardrails, source metadata, self-report comparison, and human approval gates.
